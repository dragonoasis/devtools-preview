ASCII 文字绘图 / 艺术生成器

免费在线 ASCII 艺术文字生成器。将任意文本转换为大号块字母、阴影艺术字或简易 ASCII 横幅，无需任何外部库。

什么是 ASCII 文字绘图器？

ASCII 文字绘图器使用 #、█ 和 ░ 等字符将普通文本转换为装饰性 ASCII 艺术字。每个字母在固定网格上渲染为多行字形，将普通文字变成引人注目的大型横幅。本工具内置三种风格——块状、阴影和简约，无需任何外部依赖。

如何使用 ASCII 文字绘图器

  1. 在输入框中输入或粘贴要转换的文本。
  2. 选择渲染风格：块状（5 行高的 # 字母）、阴影（带阴影效果的块字母）或简约（紧凑的 3 行字母）。
  3. 在下方输出区域查看生成的 ASCII 艺术字。
  4. 点击复制按钮将输出复制到剪贴板。

使用场景

  • README 横幅 — 为 README 文件和文档添加时尚的 ASCII 艺术标题，使项目名称和章节标题更加醒目。
  • 终端欢迎信息 — 为终端 MOTD（每日消息）屏幕、CLI 工具启动画面或服务器登录问候语生成大号文字横幅。
  • 代码注释 — 使用 ASCII 艺术注释在源代码中创建醒目的分节标记，提高代码可读性和导航性。
  • 邮件与聊天签名 — 为纯文本邮件、IRC、Discord、Slack 或其他消息平台设计独特的 ASCII 艺术签名或标志。

FAQ

支持哪些字符？
内置字体映射支持大写 A–Z、数字 0–9、空格和少量标点符号（!、.、-、_）。其他字符会被替换为备用字形。输入会自动转换为大写。
块状和阴影风格有什么区别？
块状风格使用 # 符号将每个字符渲染为 5 行高的网格。阴影风格使用相同的网格，但以 █（全块）字符渲染，并添加 ░（浅色阴影）偏移以创建 3D 深度效果。
这个工具使用 figlet.js 或其他外部库吗？
不使用。本工具使用完全自包含的字体映射，直接定义在组件中，不需要任何外部依赖或字体文件。
输出只能在等宽字体环境中使用吗？
是的。ASCII 艺术依赖于每个字符占据相同宽度。它在终端、代码编辑器、pre 标签、Markdown 代码块以及任何使用等宽（固定宽度）字体的环境中都能正确显示。
