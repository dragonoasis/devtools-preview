什么是计时器工具？

计时器工具将精密秒表和倒计时器整合在同一界面中。秒表可以毫秒级精度测量经过的时间并支持分圈记录，倒计时器则允许您设定目标时长（支持快捷预设），提供可视化进度条，并在时间到达时发出提醒。两种模式均使用 requestAnimationFrame 实现流畅、省电的显示更新。

如何使用计时器

  1. 使用顶部的标签页在秒表和倒计时之间切换。
  2. 在秒表模式下，点击“开始”开始计时，“分圈”记录分段时间，“暂停”停止计时，“重置”清除记录。
  3. 在倒计时模式下，输入小时、分钟和秒数，或选择快捷预设（1分钟、5分钟等）。
  4. 点击“开始”启动倒计时，进度条会随时间推移逐渐填满。
  5. 倒计时结束时，显示界面会闪烁红色以提醒您。
  6. 使用“暂停”可临时停止，“继续”可从暂停处恢复。

常见使用场景

  • 性能基准测试 — 测量手动操作或部署步骤所需的时间，以便比较不同优化方案的效果。
  • 番茄钟与专注时段 — 设置25分钟的倒计时进行专注工作，然后短暂休息。
  • 会议与演示计时 — 使用倒计时器控制会议节奏，或通过分圈功能为各演示环节计时。
  • 调试与性能分析 — 为浏览器或应用中的特定用户操作流程计时，在使用专业性能分析工具之前先定位慢速交互。

FAQ

秒表的精度如何？
秒表使用浏览器的 performance.now() API 和 requestAnimationFrame，提供亚毫秒级的内部精度。显示更新以屏幕刷新率（通常60帧/秒）进行，因此视觉分辨率约为16毫秒，而记录的分圈和总时间精确到微秒级别。
倒计时结束时会发出声音吗？
当前版本提供视觉提醒——显示界面会闪烁红色并显示“时间到！”的消息。由于浏览器的自动播放策略各异，视觉反馈可确保在所有浏览器中可靠地发出通知。
切换标签页后计时器还会继续运行吗？
会的。经过的时间是基于绝对时间戳（performance.now()）计算的，因此即使浏览器在后台标签页中降低动画帧率，当您返回时显示的时间仍然是准确的。
