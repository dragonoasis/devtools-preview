What Is a Favicon?

A favicon (short for favorite icon) is the small icon displayed in browser tabs, bookmarks, and mobile home screens for your website. Modern browsers and devices expect multiple sizes ranging from 16x16 pixels for browser tabs to 512x512 for PWA splash screens. This tool generates all common sizes from a single source image along with the HTML and web manifest code you need.

How to Generate Favicons

  1. Upload a source image (PNG, JPG, or SVG). Square images work best.
  2. Adjust background color, padding, and border radius to customize the output.
  3. Preview all generated sizes from 16x16 to 512x512.
  4. Download individual sizes or click Download All to get every favicon at once.
  5. Copy the HTML and manifest snippets into your project.

Common Use Cases

  • Website Branding — Create consistent favicons for browser tabs, bookmarks, and link previews.
  • PWA Icons — Generate 192x192 and 512x512 icons required for Progressive Web App manifests.
  • Apple Touch Icons — Create the 180x180 icon for iOS home screen bookmarks.

FAQ

What image format should I upload?
PNG or SVG is best for crisp results. JPG works but may introduce compression artifacts at small sizes.
Does this generate .ico files?
This tool generates PNG favicons, which are the modern standard. Most browsers support PNG favicons. For legacy .ico format, you can convert the 32x32 PNG output.
What sizes do I actually need?
At minimum: 16x16 and 32x32 for browser tabs, 180x180 for Apple Touch Icon, and 192x192 + 512x512 for PWA manifest.
